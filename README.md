# Disease Dataset EDA & Visualization

## Overview
Performed exploratory data analysis on a 5000+ record dataset to uncover trends in symptom-diagnosis relationships.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Key Insights
- Identified statistically significant correlations between symptoms and diagnosis
- Visualized data using heatmaps, distribution plots, and correlation matrices
- Improved interpretability of results

## Models Used
- Logistic Regression
- Decision Tree

## Results
- Decision Tree Accuracy: ~95%
- Logistic Regression Accuracy: ~89%
